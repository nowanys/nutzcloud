# NutzCloud的demo

## 使用方式

1. 先启动 [loach](https://github.com/nutzam/loach)
2. 然后启动2~4个CloudServiceLauncher
3. 启动 CloudWebLauncher
4. 访问页面 http://127.0.0.1:8082
